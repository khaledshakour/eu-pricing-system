## 📋 الخطوات بالتفصيل:

### الخطوة 1: رفع المشروع على GitHub

1. ادخل على https://github.com
2. اضغط على الزر الأخضر **"New"** (يسار الشاشة أعلى)
3. سمّي المشروع: `eu-pricing-system`
4. اختر **Public**
5. اضغط **"Create repository"**
6. بعد ما ينشأ المستودع، اضغط على **"uploading an existing file"**
7. اسحب **كل ملفات ومجلدات المشروع** وأفلتها في الصفحة:
   - مجلد `public/` (بداخله index.html)
   - مجلد `src/` (بداخله index.js و App.js)
   - ملف `package.json`
8. اضغط **"Commit changes"**

### الخطوة 2: ربط المشروع بـ Vercel

1. ادخل على https://vercel.com
2. اضغط **"Add New" → "Project"**
3. ستظهر لك قائمة مستودعاتك على GitHub
4. اختر مستودع **eu-pricing-system**
5. اضغط **"Import"**
6. **لا تغيّر أي إعدادات** — Vercel يكتشف تلقائياً أنه مشروع React
7. اضغط **"Deploy"**
8. انتظر دقيقة أو دقيقتين حتى ينتهي البناء ✅
9. سيعطيك رابط مثل: `eu-pricing-system.vercel.app`

### الخطوة 3: ربط الدومين الخاص بك

1. في لوحة تحكم Vercel، ادخل على مشروعك
2. اضغط **"Settings"** → **"Domains"**
3. اكتب دومينك (مثلاً: `pricing.exunion.co`)
4. اضغط **"Add"**
5. Vercel سيعطيك سجلات DNS تحتاج تضيفها عند مزوّد الدومين:

   **إذا كنت تستخدم دومين فرعي (مثل pricing.exunion.co):**
   - النوع: `CNAME`
   - الاسم: `pricing`
   - القيمة: `cname.vercel-dns.com`

   **إذا كنت تستخدم الدومين الرئيسي:**
   - النوع: `A`
   - القيمة: `76.76.21.21`

6. بعد إضافة السجلات، انتظر 5-30 دقيقة حتى يتم التفعيل
7. Vercel يضيف شهادة SSL تلقائياً (https) ✅

## 🔧 تحديث الأسعار والبنود:

لتعديل أي سعر أو بند، افتح ملف `src/App.js` وابحث عن قسم `CATEGORIES` في أعلى الملف.

مثال — لتغيير سعر الخرسانة المسلحة:
```javascript
{ name: "أعمال الخرسانة المسلحة", unit: "م³", priceRange: [1200, 2500] },
// غيّر [1200, 2500] إلى النطاق السعري الجديد
```

لإضافة بند جديد، أضف سطراً مشابهاً في القسم المناسب.
